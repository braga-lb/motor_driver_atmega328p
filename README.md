# CNC Motor Control

Projeto de CNC com controle de motores.
## ⚙️ Objetivo
Demonstrar o funcionamento de uma CNC com controle de motores, incluindo movimentação dos eixos e precisão no corte.

## 🛠️ Componentes utilizados
- Motores de passo
- Drivers de motor
- Estrutura mecânica CNC
- Microcontrolador (ex.: Arduino ou ATmega328P)

## 🚀 Como funciona
1. O microcontrolador envia sinais de controle para os drivers.  
2. Os drivers acionam os motores de passo.  
3. A CNC executa movimentos precisos nos eixos X, Y e Z.
